# Catalog

- 공개 카탈로그, 개인 카탈로그, 학습 그룹 카탈로그가 있습니다.
- 공개는 모든 사용자에게 공개되고, 개인는 학습자에게만 제공되고, 학습 그룹은 학습 그룹에 속한 학습자에게만 제공됩니다.

![Catalog](./images/catalog1.webp)

- 카탈로그를 선택하고 콘텐츠를 바로 등록할 수 있습니다.

![Catalog](./images/catalog2.webp)

!!! note "내가 등록된 학습그룹"

    학습 그룹 정보는 학습자가 입력하거나 수정할 수 없고
    파트너의 담당자가 관리합니다.
    프로필 페이지에서 내가 등록된 학습그룹 정보를 볼 수 있습니다.
