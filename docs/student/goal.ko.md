# Goal

!!! note "Competency Framework"

    역량 프레임워크에서 스킬을 선택하고, 스킬을 구성하는 요소를 목표로 설정합니다.
    설정한 목표를 위해 학습해야 하는 패스웨이 구성하고 추천하는 서비스입니다.

!!! warning "NCS, Korean National Competency Standard"

    한국에서 사용되는 NCS 역량 프레임워크만 내장되어 있습니다.

<br>
