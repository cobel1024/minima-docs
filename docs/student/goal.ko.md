# Goal

- 역량 프레임워크가 내장되어 있고, 학습 객체와 수료증, 배지와 통합되어 있습니다.

![Goal](./images/goal1.webp)

- 목표로 하는 스킬의 세부요소를 선택하고 목표로 설정합니다.

![Goal](./images/goal2.webp)

!!! note "NCS 역량 프레임워크"

    한국에서 사용되는 표준 역량 프레임워크입니다.
    직무 분류가 1478개로 되어 있고, 분류에는 직무 스킬이 13343개로 정의되어 있습니다.
    스킬에는 세부 구성요소가 47374개로 정의되어 있습니다.
    NCS를 번역하면 다른 언어에서도 직무 체계로 사용할 수 있습니다.

!!! warning "패스웨이 큐레이팅"

    목표에 맞는 콘텐츠를 패스웨이로 구성해서 추천하는 서비스입니다.
    이 기능은 구현되어 있지 않습니다.
