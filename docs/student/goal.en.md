# Goal

- Competency framework is built-in and integrated with learning objects, certificates, and badges.

![Goal](./images/goal1.webp)

- Select detailed elements of your target skill and set them as goals.

![Goal](./images/goal2.webp)

!!! note "NCS Competency Framework"

    Standard competency framework used in Korea.
    1,478 job classifications with 13,343 defined job skills.
    Skills have 47,374 defined detailed components.
    Translating NCS makes it usable as a job system in other languages.

!!! warning "Pathway Curation"

    Service that recommends content organized as pathways to match goals.
    This feature is not yet implemented.
