# Goal

!!! note "Competency Framework"

    This is a service that selects skills from the competency framework, 
    sets the elements that make up the skill as goals, configures pathways 
    that need to be learned for the set goals, and recommends them.

!!! warning "NCS, Korean National Competency Standard"

    Only the NCS competency framework used in Korea is built-in.

<br>

- Search for the skill you are targeting by selecting a category or entering it directly.

![Goal](./images/goal1.webp)

- Select the detailed elements of the target skill and set them as goals.

![Goal](./images/goal2.webp)
