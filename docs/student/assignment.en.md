# Assignment

## Step 1. Getting Started

- Agree to the compliance requirements and authenticate before starting the assignment.

![Getting Started](./images/assignment1.webp)

## Step 2. Submitting Assignment

- Submit the assignment you've completed within the deadline.

![Submitting Assignment](./images/assignment2.webp)

- Grading is in progress during the grading period.

![Submitting Assignment](./images/assignment3.webp)

!!! failure "Pre-submission Plagiarism Check"

    A plagiarism check is performed when you submit an assignment.
    If it meets the plagiarism criteria set for the assignment, you cannot submit it.

    ![Submitting Assignment](./images/assignment4.webp)

## Step 3. Grading Review

- Grading is completed using rubrics. Once grading is complete, grades are published and the grade appeal period begins.

![Grading Review](./images/assignment5.webp)

- File an appeal during the grade appeal period.

![Grading Review](./images/assignment6.webp)

## Step 4. Final Grade

- Re-grade if necessary based on the grade appeal and finalize the score.

![Final Grade](./images/assignment7.webp)
