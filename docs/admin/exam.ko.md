# Exam

!!! note "Exam Full Workflow"

    - 시험은 문제은행에서 설정한 문제 종류와 문제 수가 랜덤하게 선택되어 출제됩니다.
    - 시험 응시는 시험에서 설정한 횟수만큼 응시할 수 있습니다.
    - 재응시 시에는 이전 기록은 삭제되지 않고 활성 상태만 비활성으로 변경되어 보관됩니다.
    - 객관식과 숫자 입력 유형은 자동채점 되고, 텍스트입력, 서술형은 교강사가 채점해야 합니다.
    - 채점이 완료된면 '완료' 상태로 변경해야 학습자에게 공개되어 이의 기간이 시작됩니다.
    - 이의 기간이 끝나면 채점을 '확정'으로 변경하면 시험 프로세스가 완료됩니다.
