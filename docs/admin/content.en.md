# Content

!!! note "Content Access Permissions"

    In Minima LMS, all content requires enrollment to access.
    Content without enrollment can be searched but detailed content cannot be accessed.
    As an exception, if media content is registered in "Public Access",
    logged-in learners can view the content directly after searching.

!!! note "Create AI Quiz"

    If captions are included, click the 'Create Quiz' button to create an AI quiz.
